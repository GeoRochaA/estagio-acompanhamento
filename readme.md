# Controle de Estagio

Projeto em HTML, CSS e JavaScript para acompanhar a frequencia do estagio ao longo do ano.

A pagina permite marcar os dias em que houve estagio, registrar horario de inicio e fim, calcular automaticamente as horas por dia, por mes e no total geral. Os meses ficam recolhidos e podem ser abertos com uma seta, e a interface possui tema claro e escuro.

Os dados ficam salvos no navegador usando localStorage, entao continuam disponiveis ao recarregar a pagina.
